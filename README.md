# ProjetoPSW-Futebrol
Projeto para a matéria de PSW, desenvolvemos um programa chamado Futebrol, que serve como organizador de jogos para torneios de peladeiros.

# INSTRUÇÕES PARA O CÓDIGO:

 cd frontend;
 npm install;
 npm run json-server;

( novo terminal )
 cd frontend;
 npm run dev;



( novo terminal)
cd backend;
npm run start;
